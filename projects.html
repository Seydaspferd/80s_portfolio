  • Responsiveness for different devices

  • Integrate Google Maps

  • Integrate PDF files

  • Deploying on Netlify
  • Netlify forms with captcha
  • Hosting the Website (DNS Settings)

Hangman

In the first Javascript book I read, there was this little tutorial at the end for a very basic Hangman game. It had no lives and therefore you could not lose the game. I added lives and made the graphics for guessing wrong. I also changed the input and output with prompt and alert to a visually more appealing one directly to the DOM.


What I learned:


  • Basic game logic with Javascript
  • Manipulating the DOM with document.getElementById
  • Event listeners
  • Styling input fields
